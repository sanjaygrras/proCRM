Tasks

  • 1. admin can login by default username and password which are already insered in database by application developer. default username password policy is username - organizationName password - 12345
  • 2. admin can create new roles for the organization.
  • 3. admin can allocate permissions to roles from predefined set of Actions in DB (these Actions are basically options that will be provided to the users of system when they login as per their assined role by admin. )
  • one role (admin) and one user (organizationName with role admin) will be by default available as alredy mentioned. admin role have full access to all actions.
